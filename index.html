Firebase 연결 필요
실시간 2인 동기화를 위해 Firebase를 설정해야 해요.
무료 요금제로 충분해요 (5분이면 끝나요).
1
console.firebase.google.com 접속 → "프로젝트 만들기"
Google 계정으로 로그인 후 프로젝트 이름 입력 (Analytics 불필요)
2
왼쪽 메뉴 → 빌드 → Realtime Database → "데이터베이스 만들기"
지역: us-central1 / 보안 규칙: 테스트 모드 선택
3
⚙ 프로젝트 설정 → 내 앱 → 웹 앱 추가(</>)
앱 등록 후 나오는 firebaseConfig 객체 복사
4
ouur_app.html을 텍스트 편집기로 열어서
상단 FIREBASE_CONFIG 값을 복사한 내용으로 교체
const FIREBASE_CONFIG = {
  apiKey: "← 여기부터 교체",
  authDomain: "...",
  databaseURL: "https://...",
  projectId: "..."
};
